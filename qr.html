DEPLOY EN GITHUB PAGES (gratis):
1. Crea repo cocinero-nomada en GitHub
2. Sube toda esta carpeta (git push)
3. Settings → Pages → Branch: main / root → Save
4. En 1-2 min tendras URL: https://TU-USUARIO.github.io/cocinero-nomada/
5. Pega ese URL arriba y descarga el QR
6. Para updates: edita archivos, git push — el QR sigue funcionando
